# ui-ux-pro-max — industry-aware styling

Use for tech, product, startup, or developer portfolios.

## Direction by headline keywords

| Signal in spec | Palette / mood |
|----------------|----------------|
| engineer, developer, backend, full-stack | Dark base, monospace accents, terminal or IDE metaphor |
| designer, creative, brand | Warmer neutrals, larger type, image-forward hero |
| business, MBA, consulting | Navy/slate, restrained motion, credibility-first layout |
| student, university | Clean, approachable, education section prominent |

## Components

- Skills: tags or compact grid — not an unstyled bullet list
- Experience: timeline or left-border list — dates aligned
- Projects: 2-col grid on desktop, 1-col mobile; clear Live/GitHub CTAs

## Do not

- Mix three industry aesthetics in one page
- Use stock “corporate blue” without tying to spec personality
