# emil-design-eng — craft and micro-interactions

Use for premium, delightful, detail-oriented specs.

## Micro-interactions

- Link hover: color + underline offset or opacity — not color-only flash
- Buttons: slight scale `0.98` on active, not bounce
- Section titles: optional subtle gradient text or border accent — one technique per page

## Performance

- Prefer `transform` and `opacity` for motion
- Lazy-load heavy images only if static export allows; prefer explicit dimensions

## Restraint

- One “wow” moment (hero or projects) — not every paragraph animated
