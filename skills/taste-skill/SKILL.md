# taste-skill — anti-preset editorial UI

Use when the spec calls for bold, editorial, luxury, or non-corporate personality.

## Rules

- Ban default “AI portfolio” patterns: centered hero + three equal cards + gray subtitles
- Prefer **one strong type family** with contrast (display + mono, or serif + grotesque)
- **Asymmetric grids**: 60/40 splits, overlapping hero image, staggered project index
- Whitespace is structural — double section padding on hero and projects
- Color: one accent used sparingly (links, section labels, one CTA) — not rainbow chips

## Tailwind hints

- `tracking-tight` on headlines; `text-balance` on hero title
- Section labels: small caps or mono `text-xs uppercase tracking-widest`
- Avoid `shadow-lg` on every card — one elevated surface max
